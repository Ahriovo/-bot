# 什么是大坏蛋个人助手

大坏蛋个人助手可以理解为一款拥有丰富功能的个人专属机器人

## 我们能够做到的：

> 定时群发 循环群发 普通群发
>
> 秒抢红包 抢包通知 红包捡漏
>
> 自助头衔 付费头衔 手动头衔
>
> 自动续火 自动备注 好友消息
>
> 表情制作 趣味互动 秒赞秒评
>
> 问答系统 一键处理 撤回监控
>
> 定时任务 个性菜单 基础群管

### 以上，是我们的功能，但这不是全部

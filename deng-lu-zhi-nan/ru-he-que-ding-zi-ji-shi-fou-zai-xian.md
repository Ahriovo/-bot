# 如何确定自己是否在线

{% hint style="info" %}
**由于有时会出现假在线的情况，比如显示iPad在线但其实不在，所以这里教大家如何确认自己是否登录成功**
{% endhint %}

**点击QQ主界面上方栏目查看登录设备**

****![](../.gitbook/assets/IMG\_20220806\_180448.jpg)****

**点击退出iPad，若退出成功没有重新上线则是假在线，若退出后显示iPad重新登录则是真在线**

****![](../.gitbook/assets/IMG\_20220806\_180630.jpg)****
